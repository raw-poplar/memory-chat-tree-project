# Contributing

Thanks for contributing!

## Setup

- Python 3.10+
- Install deps: `pip install -r requirements.txt`
- Optional (faster vector retrieval): `pip install hnswlib`

## Quick sanity check

From this directory:

```bash
python -B -m compileall -q .
```

## Pull requests

- Keep changes focused and minimal.
- Include a brief description and how you tested.
